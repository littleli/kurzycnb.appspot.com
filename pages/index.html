Kurzy ČNB

Jednoduchá aplikace pro přístup k aktuálním a historickým kurzům vypsaných ČNB. A to i pomocí CORS.

CORS je pro děcka a pro lidi co používají frameworky jako Tailbone a Angular. Použití je jednoduché a následují příklady

API je jednoduše verzované, zde naleznete jen nejnovější verzi.

http://kurzycnb.appspot.com/v1/aktualni
Pro získání aktuálního kurzu (dnešní den)
http://kurzycnb.appspot.com/v1/<rok>/<měsíc>/<den>
Pro získání historického data definovaného pomocí parametrů rok, měsíc a den. Aplikace záměrně neposkytuje data před rokem 2009 a taktéž odmítne narozdíl od rozhraní ČNB vrátit výsledek pro data budoucí.
